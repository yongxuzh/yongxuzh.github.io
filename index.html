Windows环境下Pytorch安装

创建与删除环境

创建环境

环境变量名：pytorch python版本：python3.8
conda create -n pytorch python=3.8

删除环境

conda remove -n your_env_name --all

查看已有环境

conda info --envs

激活环境

Linux 系统下需要首先激活基础环境：
source activate
conda activate env_name

退出环境

conda deactivate

Pytorch安装

安装步骤

nvidia-smi查看当前GPU的所有基础信息
如果驱动版本较低，可以在Nvidia官网更新驱动
进入Pytorch官网，在pytorch环境中运行安装命令

检验安装

进入python
import torch
torch.cuda.is_available()
返回True则成功
